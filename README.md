# 📝 Wordly: A Java-Based Wordle Clone

**Wordly** is a lightweight word-guessing game built with Java. Inspired by the popular game Wordle, it challenges players to use logic and vocabulary to uncover a hidden 5-letter word within a limited number of attempts.

---

## 🚀 Features
* **Classic Mechanics:** Guess a secret 5-character word.
* **Real-time Feedback:** Get instant hints on which letters are correct and in the right position.
* **Java Core:** Built using clean, object-oriented programming principles.
* **Lightweight & Fast:** Optimized for quick sessions and easy deployment.

## 🎮 How to Play
1. The game selects a secret **5-letter word**.
2. You enter your **5-letter guess**.
3. The game provides feedback on each letter:
    * ✅ **Correct letter, correct spot:** The letter is confirmed in the right position.
    * ⚠️ **Correct letter, wrong spot:** The letter exists in the word but is in the wrong position.
    * ❌ **Incorrect letter:** The letter does not exist in the secret word.
4. Keep guessing until you crack the code!
